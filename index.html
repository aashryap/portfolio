Video app based on microservices architecture

A web based app built using NodeJs, MongoDB, ReactJS. A very simple video serving webapp. The app follows a microservices architecture rather than the traditional monolithic architecture because of which the configuration time and startup time has been reduced by more than 60 to 70 %. The microservice have been deployed in docker container. Every microservice is loosely coupled. Use of microservice reduced the difficulties that is encountered when code is transfered from development to production. Containers can be transfered easily from one place to other since they contain all the necessary libraries and dependencies. This led to reduction of setup time to a very large extend.

  • Used microservices based architecture for this project
  • 3 microservices, one for authentication based on oAuth2, second for serving video files and last for recording user pattern
  • Created REST API with NodeJS and MongoDB for serving videos and oAuth2
  • Deployed each microservices in a docker container
  • Designed frontend part of the application in the ReactJS and bootstrap
